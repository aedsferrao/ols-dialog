About the OLS Dialog

This front end to the ontology lookup service (OLS) has been developed by Harald Barsnes. (PhD student at University of Bergen, Norway).

Project web page: http://ols-dialog.googlecode.com.

More information about the OLS can be found here.

Click the help icon in the bottom left corner for help. Any further questions can be forwarded to the PRIDE team at the EBI: pride-support@ebi.ac.uk.



The platform independent graphical user interface design is based on JGoodies and has a separate license.

For licenses regarding Java see www.sun.com.

The OLS Dialog has been tested on MS Windows XP, MS Windows Vista, Linux and Mac. And should also work on older Windows versions and on other platforms, as long as Java 1.5 (or newer) is installed.
